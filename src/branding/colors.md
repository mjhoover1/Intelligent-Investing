# Signal Sentinel - Brand Colors

## Primary Palette

| Name | Hex | Usage |
|------|-----|-------|
| Sentinel Indigo (Primary) | `#4F46E5` | Buttons, highlights, logo |
| Deep Signal Blue | `#1E3A8A` | Headers, navbars |
| Signal Teal (Accent) | `#14B8A6` | Indicator values, chips, "RSI oversold" highlights |
| Alert Red | `#DC2626` | Negative alerts, drawdowns |
| Bull Green | `#16A34A` | Gains, positive indicators |
| Soft Gray | `#E5E7EB` | Borders, background elements |
| Dark Slate | `#0F172A` | Dark mode background |

## Design Principles

- **Trust**: Professional, reliable feel
- **Clarity**: Clear information hierarchy
- **Tech-forward**: Modern fintech aesthetic
- **Accessibility**: WCAG compliant contrast ratios
- **Premium**: Polished, fintech-grade look
- **Dark-mode friendly**: Works well in both light and dark themes
