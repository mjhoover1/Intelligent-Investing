## User Guides

### [Quickstart Guide](./quickstart.md)
Get up and running in 5 steps: sign up, import holdings, apply a strategy, configure Telegram, and start monitoring.

### [Rule Types Reference](./rule-types.md)
Detailed documentation of all six rule types:
- Price below/above cost percentage
- Price below/above absolute value
- RSI below/above thresholds

Includes example configurations and best practices.

### [Strategy Presets Guide](./strategies.md)
Documentation for all seven pre-built strategy presets:
- **Capital Preservation** - Loss protection
- **Swing Trader** - Momentum trading
- **Dip Hunter** - Finding buying opportunities
- **Momentum Rider** - Riding uptrends
- **Recovery Tracker** - Monitoring underwater positions
- **Long Term Holder** - Minimal alerts for buy-and-hold
- **Active Trader** - Comprehensive alerts
